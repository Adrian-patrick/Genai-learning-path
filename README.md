# Multi-Agent System Roadmap  
LangChain · LangGraph · Langfuse

---

## Levels

**Level 1 — Basic Agent (LangChain)**  
- Single agent, tool usage, prompt basics  

**Level 2 — Structured Prompting**  
- JSON outputs, templates, reliability  

**Level 3 — LangGraph Orchestration**  
- State, nodes, flow control  

**Level 4 — Multi-Agent System**  
- Planner + worker, task delegation  

**Level 5 — Memory Systems**  
- Short-term and long-term memory  

**Level 6 — MCP + Tools**  
- MCP architecture, external tools  

**Level 7 — Observability (Langfuse)**  
- Tracing, prompt versioning, evaluation  

---

## Final

**Production System**  
- End-to-end design, failure handling, optimization  

---

**Flow:**  
Single Agent → Multi-Agent → Production System
